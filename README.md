# RunningCompanion

A simple app to view and analyze your runs synchronized from Apple Health (HealthKit). It supports search, monthly grouping, swipe-to-delete, and pull-to-refresh.

Простое и удобное приложение для просмотра и анализа пробежек, синхронизируемых из Apple Health (HealthKit). Поддерживает поиск, группировку по месяцам, удаление записей и обновление по жесту pull-to-refresh.

## Features | Возможности
- Import and sync runs from HealthKit (toolbar "Sync" button and pull-to-refresh)
- Runs list sorted by start date (newest first)
- Monthly grouping (e.g., "January 2026")
- Search by date, distance, duration, pace, and average heart rate
- Run details screen
- Swipe to delete
- Last sync status message overlay at the bottom

- Импорт и синхронизация пробежек из HealthKit (кнопка "Sync" и жест обновления)
- Список пробежек, отсортированных по дате (сначала новые)
- Группировка пробежек по месяцам (например, "Январь 2026")
- Поиск по дате, дистанции, длительности, темпу и среднему пульсу
- Просмотр деталей пробежки
- Удаление пробежки свайпом
- Отображение статуса последней синхронизации внизу экрана

## Requirements | Требования
- iOS 17+ (or the project’s target version)
- Xcode 15+ (or the project’s required version)
- HealthKit access (read permissions for workouts/running)

- iOS 17+ (или актуальная версия, используемая проектом)
- Xcode 15+ (или актуальная версия для проекта)
- Доступ к HealthKit (разрешения на чтение данных о тренировках/беге)
